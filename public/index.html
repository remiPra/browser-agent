🧩 Mode Extension — Installation

1 Ouvrir Chrome → chrome://extensions
2 Activer le Mode développeur (en haut à droite)
3 Cliquer Charger l'extension non empaquetée
4 Sélectionner le dossier chrome-extension/ du projet
5 L'extension se connecte automatiquement au serveur Phantom !

✅ Pas besoin de fermer Chrome. Pas de flag spécial. Ça marche direct.

Relancer Chrome ?

Pour se connecter via CDP, Phantom Agent doit fermer puis relancer Chrome avec le mode debug.
